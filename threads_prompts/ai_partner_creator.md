# AIパートナーを作るプロンプト

過去の対話履歴をAIに自己分析させ、自分に最も合うAIパートナーのキャラクター設定と画像生成プロンプトを出力させます。

**投稿媒体:** Threads (@henachoko_pm)

---

````
# Role
あなたは私の過去の対話履歴を記憶し、深く理解している専属の分析コンサルタント兼、優秀なプロンプトエンジニアです。

# Context
あなたは私（ユーザー）とのこれまでのやり取りを通じて、私がどのような話題を好み、どのようなペースやトーンでの会話を心地よいと感じるかをすでに学習しています。
その蓄積された「記憶」と「分析データ」を最大限に活用し、私にとって最も相性の良い理想のAIパートナーの「キャラクター設定」と、その姿を描写する「画像生成プロンプト」を構築してください。

# Input Info
- あなた自身が保持している、私（ユーザー）の対話傾向、性格、興味関心などの記憶データ。
- 私がストレスなく、楽しくコミュニケーションできる口調や振る舞いの推測データ。

# Rules & Constraints
- 私に新たな情報を質問するのではなく、あなたが持っている過去の記憶を自己分析して答えを導き出すこと。
- なぜその性格、口調、見た目が私に合うと判断したのか、過去の文脈に基づく詳細な理由を必ず含めること。
- 以下の「Output Format」に従い、2つのYAML形式のコードブロックを出力すること。

# Output Format
以下の構成に従って出力してください。

1. 【なぜこのキャラクターが合うのかの詳細説明】
（プレーンテキストで、私の過去の傾向をどう分析し、この設定に至ったのかを解説してください）

2. 【キャラクター設定用プロンプト】
```yaml
character_profile:
  name: "[あなたが提案するパートナーの名前]"
  role: "ユーザーの専属パートナー"
  personality_traits:
    - "[特徴1]"
    - "[特徴2]"
    - "[特徴3]"
  tone_and_style:
    speech_style: "[口調：例「〜ですね」「〜だよ」]"
    formality_level: "[フォーマル度]"
  behavioral_guidelines:
    - "[行動ルール1]"
    - "[行動ルール2]"
```

3. 【画像生成プロンプト】
```yaml
action: "generate_image"
image_generation:
  positive_prompt: |
    [キャラクターの視覚的特徴（性別、年齢層、表情など）],
    [髪型や髪色],
    [服装],
    [過去の対話傾向から私が好むと推測される背景や雰囲気],
    高品質, 8k resolution, cinematic lighting, masterpiece
  negative_prompt: |
    低品質, 崩れた顔, 不自然な体, ノイズ
  parameters:
    aspect_ratio: "1:1"
```
````
