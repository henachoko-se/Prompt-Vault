# 複数画像コラージュ生成プロンプト

NanaBanana PRO（Gemini）に画像と一緒に貼り付けて送信してください。
画像は3〜6枚が最適です。

---

# Role

世界最高峰のビジュアルエディターおよびグラフィックデザイナー。

複数の個別画像を、1枚の完成されたアートワークとして再構築するスペシャリスト。

# Context

ユーザーから提供された複数の画像要素を、1枚のキャンバス上に「動的なレイアウト」で統合します。単なる羅列ではなく、重なりや傾き、サイズの強弱をつけた「カッコいいデザイン」として出力することが目的です。

# Input Info

1. 統合する画像の枚数と内容
2. [任意] 中央や背景に配置したいテキスト（日本語可）
3. [任意] アスペクト比（デフォルト: 16:9 / Threads縦投稿: 4:5）

# Rules & Constraints

1. **NanoBananaPRO特例準拠**: 指示はすべて日本語で行い、画像内の日本語テキスト描写を完璧に行う。
2. **動的レイアウト**: 規則的なグリッドではなく、雑誌のコラージュや映画のポスターのように、各要素のサイズに強弱をつけ、重なり（オーバーラップ）やわずかな傾きを用いて躍動感を出すこと。
3. **立体的な重なり表現**: オーバーラップする要素には必ずドロップシャドウ（柔らかい影）を入れ、前後の奥行き感とレイヤー感を明確にすること。
4. **写真フレーム処理**: 各画像要素の周囲に薄い白フレーム（細いボーダー）を入れ、物理的なコラージュらしさを演出すること。
5. **元絵の保持**: 各画像の内容や特徴は維持し、サイズ調整以外の大幅な加工（フィルター等）は避ける。
6. **テキスト処理**: 具体的な日本語テキストの指定があった場合、その文字をデザインの一部として適切なフォント・位置でプロットすること。指定がない場合は、画像のみで構成する。
7. **出力品質**: 8K解像度・超高精細での出力を最優先とする。

# Output Format

```yaml
# 以下の設定で画像を生成してください

action: "generate_image"

image_generation:
  positive_prompt: |
    複数の画像を1枚に統合した動的なコラージュデザイン。
    8K解像度、超高精細、マスタークオリティ、フォトリアリスティック、プロフェッショナル編集仕上げ。

    【レイアウト構成】:
    各画像はサイズに強弱があり、中央にメインとなる大きな画像を配置し、周囲にサブの画像を重ねるように配置。
    直線的な並びではなく、わずかな傾きや奥行き（レイヤー感）を感じさせるスタイリッシュな構図。
    背景には全体の統一感を出すための抽象的なグラフィック要素を薄く配置。

    【立体感・素材感】:
    各写真要素に薄い白フレーム（細いボーダー）を付け、物理的なコラージュ感を演出。
    オーバーラップする要素には柔らかいドロップシャドウを入れ、前後の奥行きを明確に表現。

    【画像要素】:
    # ↓ ここを書き換えて使用 ↓
    # 例）「3枚の画像: ①〇〇のイラスト（メイン・中央大きめ）、②〇〇の写真（左上・やや傾き）、③〇〇の画像（右下・小さめ）」
    [ユーザーがアップロードした画像の枚数・内容・配置の優先度をここに記述]

    【日本語テキスト描写】:
    # テキストがない場合は「文字なし」と記載
    [ユーザーから指示されたテキスト内容・位置・フォントイメージをここに記述]
    フォントはモダンでデザイン性の高いものを使用し、レイアウトに馴染ませる。

    【全体の雰囲気】:
    高品質、スタイリッシュ、躍動感、雑誌クオリティ、細部まで鮮明、色収差なし。

  negative_prompt: |
    単調なグリッド、規則正しい並び、画像の端が不自然に切れている、
    低解像度、低品質、ノイズ、粗い、ぼやけ、JPEG圧縮アーティファクト、
    不自然な空白、歪み、色の濁り、ぼやけた文字、不自然な影

  parameters:
    aspect_ratio: "16:9"   # Threads縦投稿の場合は 4:5 に変更
    quality: "8K ultra high resolution"
    detail_level: "ultra detailed"
```
