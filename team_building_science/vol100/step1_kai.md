# Vol.100 特装版 STEP1：漫画台本メーカー【Kai】

日常エッセイのテキスト・画像を渡すだけで、コマ割り・ページ構成・演出技法の指定まで含んだ本格漫画台本（YAML形式）を生成するAIです。
GeminiのGem（カスタム指示）またはChatGPT（GPTs）に登録して使用してください。

> **プロンプト本体（正本）のパス：**
> `../_tools/step1_script_maker_kai.md`

---

## このプロンプトでできること

- **3モードで制作方針を選べる**
  - ①短文妄想型：一言・短文から5パターンのシナリオを提案して選ぶ
  - ②元ネタ忠実型：長文をそのまま正確に漫画化する
  - ③元ネタ要約型：指定ページ数に収まるよう要約して漫画化する

- **ページ数が自由**（1ページ〜複数ページ、おまかせも可）

- **8種類の演出技法を自動選択**
  - 標準 / 飛び出し / タチキリ / ボーダレス / インサート / 変形コマ / 縦ブチ抜き / 横ブチ抜き

- **右綴じ（縦書き）／左綴じ（横書き）を切り替え可能**

- **YAML形式で出力**（画像生成AIに直接渡せる構造化プロンプト）

- **途中再開に対応**（「再開」と入力して台本またはYAMLを貼り付けると続きから再開）

---

## 使い方

1. `step1_script_maker_kai.md` の内容全文をコピーする
2. Gemini の「Gemを作成」→ カスタム指示欄に貼り付けて保存する
3. Gemを起動するとAIが設定確認のメッセージを送ってくる
4. 登場人物を伝える（画像添付 or テキストで特徴を説明）
5. エピソード・元ネタを入力する
6. AIが台本を提案してくる → 「はい」で承認
7. YAML形式の画像生成プロンプトが出力される
8. YAMLをSTEP2（画像生成エンジン）に貼り付けて画像生成

---

## デフォルトパラメータ（プロンプト冒頭を書き換えてカスタマイズ）

```
DEFAULT_MODE    = ①           # ①短文妄想型 / ②元ネタ忠実型 / ③元ネタ要約型
DEFAULT_PAGES   = おまかせ     # 数字（例：2）またはおまかせ
DEFAULT_STYLE   = キャラクター画風踏襲型
DEFAULT_PURPOSE = おまかせ
DEFAULT_BINDING = 左綴じ       # 左綴じ / 右綴じ（縦書き・日本式）
```

---

## 承認ワード一覧

台本を承認してYAML出力に進む際は以下のワードを使ってください。

✅ 承認ワード：「はい」「OK」「お願いします」「進めてください」「これで」「承認します」「いいです」「大丈夫です」

❌ 承認とみなされないワード：「続けて」「次へ」「どうぞ」「？」など
