# Vol.100 インフォグラフィック：デフォルトパラメータのカスタマイズ

NanaBanana（Gemini）に貼り付けて画像生成してください。
へなPのキャラクター画像を添付して送信すると外見が安定します。

---

```
横長（16:9）のインフォグラフィック画像を生成してください。

【全体レイアウト】
- 背景：パステルブルー（淡い青色）
- 上部：タイトル帯（濃紺背景・白文字）
- 中央：パラメータ一覧（2グループに分けて並べる）
- 下部：へなPとひとことコメント

【タイトル帯（上部）】
大タイトル（白文字・太字）：「⚙️ デフォルトパラメータのカスタマイズ」
サブタイトル（白文字・細字）：「プロンプト冒頭の値を書き換えるだけで毎回の設定を省略できます」

【中央：パラメータ一覧】

3列構成のカードを2グループに分けて表示する。
左列：日本語の項目名（大きめ・太字）、中列：パラメータ名（等幅フォント風・小さめ）、右列：設定値と説明。

■ グループA「基本設定」（薄い黄色の背景帯でグループ化）

カード①
左：出力モード
中：DEFAULT_MODE
右：① 短文妄想型 / ② 忠実型 / ③ 要約型

カード②
左：出力ページ数
中：DEFAULT_PAGES
右：おまかせ（数字で固定も可）

カード③
左：画風設定
中：DEFAULT_STYLE
右：キャラクター画風踏襲型 / タッチ名を直接入力

カード④
左：綴じ方向
中：DEFAULT_BINDING
右：右綴じ（縦書き）/ 左綴じ（横書き）

■ グループB「出力安定化」（薄いオレンジの背景帯でグループ化・右上に「うまく出ない時はオフに！」バッジ）

カード⑤
左：セリフ表示
中：DEFAULT_SERIF_MODE
右：オン（セリフあり）/ オフ（吹き出しを空白に）

カード⑥
左：コマ配置
中：DEFAULT_LAYOUT_MODE
右：オフ（AIに任せる）/ オン（座標を詳細指示）

カード⑦
左：特殊コマ
中：DEFAULT_SPECIAL_PANEL
右：オン（全技法あり）/ 変形コマオフ / 飛び出しオフ / 両方オフ

カード⑧
左：コマ数設定
中：DEFAULT_PANELS / MIN / MAX
右：基本4コマ・最小3・最大5（数字で変更可）

カード⑨
左：YAML送信
中：DEFAULT_YAML_SEND_MODE
右：1枚ずつ（全ページを別ブロックで一括出力）/ まとめて（全ページを1ブロックにまとめて出力）

【カードの共通スタイル】
- 角丸・薄い影でカード風に
- 日本語の項目名を最も大きく太字で表示すること
- パラメータ名は等幅フォント風（コードっぽく）で小さめに添える
- グループAとグループBは背景色で明確に区別する
- グループBの「うまく出ない時はオフに！」バッジはオレンジ色で右上に小さく

【下部：へなPとひとことコメント】
- へなPが得意げな表情（添付画像参照）
- 吹き出しで「冒頭の数字を変えるだけ！まずはSPECIALから試してみて」と言っている
- アニメ調のちびキャラスタイル

【デザイン指定】
- 全体的にポップでかわいらしいスタイル
- グループAとグループBの色の対比を明確に
- 日本語テキストはすべて正確に表示すること
- パラメータ名は特に大きく読みやすく配置すること
```
