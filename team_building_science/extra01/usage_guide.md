# 図解しおり Pro版 使い方ガイド

このガイドは、購入後にはじめて『図解しおり』Pro版を使う方向けの入力例です。難しい書き方は不要です。まずは長文をそのまま貼り付けてください。

## まず入れる基本の一文

```
以下の文章を読んで、図解にした方がよい場所を重要度順に判定してください。
note本文で使う想定なので、スマホで読みやすい比率も判断してください。

（ここに本文を貼る）
```

Threads投稿をまとめたい場合は、次のように入れます。

```
以下のThreads投稿案を読んで、1枚目に置くと伝わりやすい図解と、
本文途中で補足するとよい図解を判定してください。

（ここに投稿案を貼る）
```

仕事・家庭文書に使う場合は、用途を一言足すと精度が上がります。

```
以下はPTAのお便りです。
保護者がスマホで見たときに、保存しておきやすい図解候補を判定してください。

（ここに本文を貼る）
```

```
以下はPM向けの会議資料です。
論点が混ざって見える箇所を、図解で整理できるか判定してください。

（ここに本文を貼る）
```

## 図解しおりマップが出た後の使い方

図解しおりは、最初に「どこに図解を入れるか」のマップを出します。その後、必要に応じて生成方法を選びます。

### 1枚だけ作る

最優先の図解だけ試したいときに使います。

```
1枚だけ生成してください。
```

または、候補番号を指定しても大丈夫です。

```
2番だけ生成してください。
```

### 全部まとめて進める

候補をまとめて作りたいときに使います。

```
全部生成してください。
```

長文で候補が多い場合、画像生成には時間がかかります。複数枚を作るときは、数分かかる前提で見てください。

### 番号指定で必要なものだけ作る

候補の中から、必要なものだけ選べます。

```
1番と3番を生成してください。
```

```
有料エリア前に置く図解だけ生成してください。
```

### 生成しないでプロンプトだけ使う

別の画像生成AIや、あとでChatGPTに投げ直したい場合に使います。

```
生成せず、画像生成用プロンプトだけ使います。
```

## 修正依頼の例

生成後に気になるところがあれば、自然な言葉で修正できます。

### 配色を変える

```
2番をもう少し明るく、note向けにやわらかい色にしてください。
```

```
仕事資料っぽく、白・濃紺・グレー中心にしてください。
```

### 図解内テキストを短くする

```
文字が少し多いので、スマホで読めるように短くしてください。
```

```
カード見出しをもっと短い言葉にしてください。
```

### 型を変える

```
3ポイント型ではなく、Pros/Cons型でもう一度組み直してください。
```

```
比較表ではなく、判断フロー型で試してください。
```

### 比率を指定する

```
note本文に入れたいので、2:3の縦長で作ってください。
```

```
Threadsの1枚目にしたいので、4:5で作ってください。
```

## うまく使うコツ

- 長文を貼る前に、用途を1行だけ添える
- note本文、Threads、スライド、PTA文書など、置き場所を伝える
- 「読者に何をしてほしいか」を一言添える
- 画像内に入れたい固有名詞があれば先に伝える
- キャラクターや配色の希望がある場合は、最初に伝える

## 期待値の目安

『図解しおり』Pro版が得意なのは、画像生成そのものの待ち時間を短くすることではありません。

長文を読んだあとに、

- どこを図解すべきか
- 何枚くらい必要か
- どの型が合うか
- どの比率が読みやすいか
- 図解内の文字をどこまで短くするか

を先に整理することです。

画像内の日本語は、利用環境や画像生成AIの状態によって崩れることがあります。必要に応じて、Canvaなどで文字だけ後から整える使い方もおすすめです。
