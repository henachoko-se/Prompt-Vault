## ■ これまで試した解決策、こうじゃありませんでしたか？

```
❌ 無料のChatGPTに「図解して」と頼んだ
   → 文字化け・構造が破綻した謎の絵が返ってくる

❌ 画像生成系の図解プロンプトを試した
   → 結局「どの型にするか」は自分で決めなきゃいけない

❌ 「8型のチートシート」を覚えた
   → 業務で使う複雑な構造（ロジックツリー・スイムレーン・ピラミッド等）に対応できない

❌ デザインツール（Canva・Figma）でテンプレを探した
   → 「テンプレを見つける時間」と「型を選ぶ時間」が両方かかる

❌ プロのデザイナーに外注した
   → 1枚3,000〜10,000円。月10枚で予算が溶ける
```

問題の本質は、「絵を描かせる」ことじゃありません。

**「どの型にするか」を、AIに自分で判断させる仕組み** が、これまで存在しなかっただけです。

## ■ 3ヶ月かけて整理した、12大分類 × 68小パターン

PMの仕事で使う図解、PTAのお便り、自由研究、社内資料──ぜんぶ眺めて気づきました。

世の中で使われている図解パターンは、整理すれば **12の大分類 × 68の小パターン** に収まります。

```
要約型（5）／比較型（8）／手順型（4）／構造型（10）
分類型（7）／関係型（5）／因果型（5）／判断型（5）
変化型（4）／データ型（6）／ストーリー型（5）／CTA型（4）
```

そして、これらを **4つの判断軸 × 7つの選択ルール** で自動的に振り分けるロジックも作りました。

```
【4つの判断軸】
A. 情報の関係性（順序／対比／分類／因果／構造）
B. 読者の状態（認知前／検討中／課題感／行動準備）
C. 情報量（1メッセージ／2〜4／5〜7／8個以上）
D. 伝えたい目的（理解／納得／共感／比較／選択／保存／行動）

【7つの選択ルール】
順序語が多い → 手順型
比較語が多い → 比較型
悩み・課題語が多い → 因果型（問題→原因→解決）
種類・分類語が多い → 分類型
時間語が多い → 変化型（タイムライン）
数字が多い → データ型
キャラ・関係語が多い → 関係型（相関図）
```

この体系を、AIに丸ごと積み込んで、**長文を貼るだけで自動選定するエージェント** に仕立てたのが、**「図解しおり」Pro版** です。

## ■ 認知負荷を肩代わりする、という効果

ここまで読んで「で、これが何に効くの？」と感じる方もいると思います。

結論から言うと、**読み手の脳の負荷を下げる効果** が一番効きます。

文章しかない資料は、読み手が一文一文を読んで、頭の中で構造化しなければなりません。
これは認知科学で **「認知負荷（Cognitive Load）」** と呼ばれる負担で、上司・経営層・忙しいママ友など"時間がない読み手"が真っ先に飛ばすのが、この負荷の高い文書です。

図解は、この負荷を肩代わりする道具です。
**ただし、正しい場所に正しい型が置かれていないと、逆に負荷が上がる。**

「図解しおり」Pro版は、この"場所と型のマッチング"だけを最適化します。
画像生成は他のAIに任せていい。**人間が長く悩んでいた"上流の判断"だけ** を、丸ごと引き受ける設計にしました。
