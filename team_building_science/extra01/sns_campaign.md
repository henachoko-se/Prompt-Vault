## Day 1：ペインの顕在化

**目的:** 「図解の型を選べない苦痛」を言語化し、共感で足を止める

### 漫画シナリオ（4コマ）

**コマ1:**
Slackに通知。上司「この仕様書、分かりやすく図にしといて。明日の朝イチで使うから」

**コマ2:**
ファイルを開く。文字がぎっしり詰まった仕様書。主人公、固まる。

**コマ3:**
PowerPointを開いたまま20分フリーズ。吹き出し「フロー図？　マトリクス？　比較表？　そもそも何が言いたい文章なんだこれ……」

**コマ4:**
深夜のオフィス。図形を1つも置けていない画面。「図の"種類"が決まらないと、1ミリも動けない」

### Threads 投稿文

```
「この仕様書、図にしといて。明日の朝イチで使うから」

このメッセージを受け取った瞬間の、あの止まり方——

フローチャートか。
マトリクスか。
比較表か。

「どの図にするか」だけで、毎回30分が消えていました。

図を描く技術より先に、「何の図を描くか判断する技術」が足りていなかったんだと思います。

明日、続きを書きます。
```

## Day 2：旧解決策の否定

**目的:** 「AIに頼めばいい」という逃げ道を塞ぎ、問題の深さを体感させる

### 制作準備（必須）
- ChatGPT / Gemini などに長文仕様書を投げて、意図的に「破綻した図解」を生成しスクショ撮影

### 漫画シナリオ（4コマ）

**コマ1:**
主人公、閃く。「そうだ——AIに図解させれば解決じゃん！」

**コマ2:**
長文仕様書をChatGPTにコピペ。「Enter——！」期待に満ちた表情。

**コマ3:**
← **ここは漫画コマではなく、実際の失敗スクショを挿入する（リアリティが命）**

**コマ4:**
ズコーッとずっこける。「……俺が欲しいのは綺麗な飾り絵じゃない。"論理の構造"なんだよ！」

### Threads 投稿文

```
「AIに図解させれば解決する」と思って、長文をそのまま投げてみました。

出てきたのがこれです↓

（スクショ添付）

AIは「絵を描く」ことはできます。
でも「この文章、フロー型かマトリクス型か」を文脈から推論して選ぶ、ということができない。

「図解を自動化するには、型の判断を先にやらせる必要がある」という結論に、ここで気づきました。

明日、その話を書きます。
```

## Day 3：パラダイムシフト・チラ見せ

**目的:** 解決策の存在を示唆し、「残りは？」という渇望（ツァイガルニク効果）を刺激する

### 使用画像
- 8型パターン画像のうち2枚（**対比が際立つ組み合わせ推奨**：比較型＋ステップ型 など）

### 漫画シナリオ（4コマ）

**コマ1:**
絶望する主人公のそばに、ベテランPM登場。「型を覚えようとするから迷うんだ。判断をAIに渡せばいい」

**コマ2:**
「PMが仕事で使う図解パターン、実は8種類しかない。例えばこの2つだ」
← **漫画コマ内またはすぐ下に、パターン画像2枚を配置**

**コマ3:**
主人公「これを覚えれば……？」ベテラン「覚えなくていい。長文を投げると、AIが8型の中から自分で選んで出力する仕組みを作った」

**コマ4:**
主人公「AIが"型の判断"まで……？」目が光る。

### Threads 投稿文

```
PMが仕事で使う図解パターン、ぜんぶで8種類しかない。

3ヶ月かけて整理した結論です。そのうち2つ、見てください。

（画像2枚添付）

「覚える」じゃなくて「AIに選ばせる」という発想で、この問題が解けるかもしれない——

残り6パターンは、明後日まとめて出します。
明日はこの仕組みが本当に動くか、実際の変換を見せます。
```

## Day 4：圧倒的証拠（Before / After）

**目的:** 理屈ではなく「視覚的な落差」で購買意欲を最高潮に引き上げる

### 制作準備（必須）
- **Before素材:** 実際の長文テキスト（仕様書・議事録・要件定義など40〜60行）
- **After素材:** 図解しおりを使って生成した実際のスクショ（Thinkingの推論過程が見えると◎）
- **使用画像:** 8型パターン画像のうち最も視覚インパクトの強い1枚

> **Day 4は漫画なし。実際のスクショだけで語る。**
> 漫画で「すごかった！」と描いても疑念を招く。証拠だけが読者を動かす日。

### Threads 投稿文

```
Before：47行の要件定義テキスト

After：↓

（図解画像添付）

「ここはステップ型、ここはマトリクス型」と、AIが文脈から自己判断して型を選び、図解まで出力しています。

「どの型にするか」で悩む30分が、消えました。

明日、これを動かすプロンプトをnoteで有料公開します。
PMの残業1時間分より安い価格に設定するつもりです。
```

## Day 5：全貌公開・販売開始

**目的:** 「今すぐ手に入れたい」に変換し、noteへ誘導する

### 制作準備（必須）
- **コラージュ画像:** 残り3パターンを含む全6パターン一覧（Day 3・4で未公開のもの3枚を束ねる）

### 漫画シナリオ（1〜2コマ）

**コマ1:**
晴れやかに定時退社する主人公。「図解の型で迷うことが、なくなりました」

**コマ2（任意）:**
スマホにnote公開の通知。「出しました」

### Threads 投稿文

```
公開しました。

（コラージュ画像：全8パターン一覧）

「図解しおり」——

長文に🔖を挟むように、AIが図解の挿入場所と型を全部マークします。
8種類のパターンを文脈から自己判断し、画像生成プロンプトまでセットで出力します。

「どの図にするか問題」に悩む時間が、一生なくなります。
PMの残業1時間分より安い価格です。
```
