# 【図解1】通常ChatGPT相談との差分図「できること／固定すること」

**挿入位置：** 「普通のChatGPTでも、図解の型は選べます」セクション末尾
**目的：** 通常のChatGPTでも型提案はできることを認めたうえで、図解しおりGPTsが固定する判断範囲を見せる
**形式：** 横長 16:9 / 比較インフォグラフィック

> ※ 既存AIを悪く見せる素材にしない。読者はChatGPT課金ユーザーの可能性が高いため、「普通のChatGPTでもできること」と「GPTsで毎回揃えること」を冷静に比較する。

```
横長（16:9）の比較インフォグラフィック画像を生成してください。

【全体レイアウト】
- 背景：白
- 上部：ネイビーのタイトル帯
- 中央：左右2カラム比較
- 下部：マスタードの結論帯

【タイトル帯】
大タイトル：「ChatGPTは使える。だから判断手順をGPTsに固定する」
サブタイトル：「毎回迷う、場所・型・比率・文字量を揃える」

【左カラム：通常のChatGPT相談】
- 見出し：「普通のChatGPT」
- アイコン：チャット吹き出し
- 箇条書き：
  1. 用途を言えば型は提案できる
  2. 画像生成もできる
  3. ただし毎回、指示の組み立てが必要
  4. 候補の優先順位や比率は自分で詰めがち

【右カラム：図解しおりGPTs】
- 見出し：「図解しおりGPTs」
- アイコン：しおりマーカー付きの文書
- 箇条書き：
  1. 68パターンから重要度順に選定
  2. 挿入位置・型・比率をマップ化
  3. 図解内テキストを短く圧縮
  4. 生成前に「1枚だけ／全部／番号指定」を確認

【下部：結論帯】
中央メッセージ：「差は、AIの賢さではなく“図解前の判断手順”を固定できること。」

【デザイン指定】
- ネイビー（#1A2B5C）／白／シルバーグレー／マスタード（#E2A93D）
- どちらかを貶す表現にしない
- テキストはスマホでも読めるサイズにする
- 左右の情報量を揃え、右側だけ過剰に盛らない
- 横長16:9を厳守
```

# 【図解3】Before/Afterデモ「長文 → 図解しおりマップ＋画像生成」

**挿入位置：** 「実際の出力イメージ」セクション末尾
**目的：** 長文を投げると「No.〇〇 〇〇型」「推奨アスペクト比」「図解密度」「画像生成プロンプト」まで返るWowを視覚化
**形式：** note本文向けは縦長 2:3 または 3:4 / スライド転用向けのみ横長 16:9

> ※ 可能なら「実際に図解しおりPro版を動かしたスクショ」を使うのがベスト（Day4のSNS素材と兼用可）。
> モックアップで代替する場合は、まず以下の **縦長版** を使用。noteスマホ読者に見せる販売ページでは、横長左右比較より縦長の方が読みやすい。

```
縦長（2:3）のインフォグラフィック画像を生成してください。

【全体レイアウト】
- 背景：白
- 上部：タイトル帯（ネイビー背景・白文字）
- 中央上：Before（長文の塊）
- 中央：図解しおりPro版の解析フロー
- 中央下：After（マップ＋推奨比率＋生成イメージ）
- 最下部：成果メッセージ帯（マスタード背景・白文字）

【タイトル帯】
大タイトル（白文字・太字）：「長文 → あとから見返せる図解設計へ」
サブタイトル（白文字・細字）：「68パターン選定・縦横比判定・画像生成プロンプトまで」

【中央上：Before】
- ラベル：「Before：47行の要件定義テキスト」（グレー帯）
- A4文書風の白い紙を配置し、本文は読めない細いダミーラインで高密度に表現

【中央下：After】

- 「No.〇〇 〇〇型」「2:3 縦長」「Showcase」が視認できるサイズにする
- BeforeとAfterの差が分かるように、After側はしおりマーカーと結論帯を強調
- 横長3カラムの単調なカード並びにしない
```

# 【図解4・任意】比較インフォグラフィック「図解が刺さらない真の原因は"場所"」

**挿入位置：** 「認知負荷を肩代わりする、という効果」セクション末尾（必要に応じて）
**目的：** 「デザイン vs 場所」という問題の切り分けを瞬時に伝える補助図解
**形式：** 横長 16:9 / 左右比較インフォグラフィック

> ※ 図解4は補助。記事の長さやスクロール感を見て、入れすぎだと判断したら省略してOK。

```
横長（16:9）のビジネスインフォグラフィック画像を生成してください。

【全体レイアウト】
- 背景：白
- 上部：タイトル帯（ネイビー背景・白文字）
- 中央：左右2分割
- 下部：結論メッセージ帯（マスタード背景・白文字）

【タイトル帯】
大タイトル（白文字・太字）：「図解が刺さらない真の原因」
サブタイトル（白文字・細字）：「デザインじゃない。"場所と型のミスマッチ"です。」

【中央：左右比較】

■ 左側（薄いグレー背景・「思い込み」ラベル）
カードタイトル：「みんなが思っている原因」
チェック項目（グレー文字・▲マーク）：
- ▲ デザインのセンスが足りない
- ▲ 画像生成プロンプトが下手
- ▲ ツールの使い方が分からない
キャプション（小さく）：「ここに労力を注いでも、刺さらない図解は変わらない」

■ 右側（明るい白背景・ネイビー強調枠・「真の原因」ラベル）
カードタイトル：「実際の原因（9割がここ）」
チェック項目（ネイビー文字・⭕マーク）：
- ⭕ 入れる場所のミスマッチ
- ⭕ 文章構造に合わない型を選んでいる
- ⭕ 上流の"設計判断"を人間がやっている
キャプション（強調）：「= 『図解しおり』Pro版が解決する領域」

【下部：結論帯】
横長の帯（マスタード背景・白文字・太字）
中央メッセージ：「図解は"作る"より"場所と型の判断"が9割。」

【デザイン指定】
- 左右の対比は穏やかなグレースケール vs ネイビー強調で表現
- ビジネス資料に貼っても違和感のない硬めのトーン
- 角丸・薄い影・余白を十分に取る
- 日本語テキストはすべて正確に表示
```
