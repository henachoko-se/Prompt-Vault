# Vol.102 図解プロフィールカード出力フロー図解プロンプト（Vol.90応用編・4ステップ）

Vol.90のプロンプトを使って、実際に画像が出力されるまでの流れを初心者向けに図解するプロンプト群です。
`infographic_steps.md`（STEP 1〜6：汎用AI使い方／基本編＋応用編）の続きに配置する想定。
アスペクト比はすべて **16:9**。

**スタイル共通方針（全プロンプト共通）：**
- キャラクター・人物・吹き出し・漫画コマ・セリフ・ちびキャラは一切使わない
- スマートフォンの画面モックアップ＋ステップ番号＋矢印・ハイライト枠のみで構成する
- フラットデザイン・UI解説書スタイル（Appleのサポートページや操作ガイドに近い印象）
- 配色：ブランドブルー（#0056D2）・白・グレーを基調。アクセントはオレンジ（#FF6B35）のみ

**対応記事:** [article.md](./article.md)

**前提：** Vol.102の汎用STEP 1〜6を終えて、Vol.90のプロンプトをGemini/ChatGPTに貼り付け済みの状態からスタート。

## FLOW 4：YAMLを貼り付けて画像生成

```
note記事本文に埋め込む操作手順の図解画像（アスペクト比 16:9）。
フラットデザイン・UI解説書スタイル。キャラクター・人物・漫画要素は一切使わない。
配色：ブランドブルー（#0056D2）・白・ライトグレーを基調。強調色はオレンジ（#FF6B35）のみ。

テーマ：「FLOW 4：YAMLを貼り付けて画像生成」

全体レイアウト（横長 16:9）：
左端1/5にFLOWタイトルエリア。残り4/5に3つのサブステップを横並びで配置し、各サブステップ間をオレンジの「→」矢印でつなぐ。

【左端エリア】
- ブランドブルーの縦帯
- 白文字で「FLOW」（小・上）と「4」（大・中）
- 白文字で「貼り付けて\n画像生成」（小・下）

【サブステップ①】
- ライトグレーの角丸カード
- 上部：オレンジの丸に白数字「①」
- 中央：スマートフォン正面モックアップ
  - 画面内：新しいチャットの入力欄にYAMLの長いテキストが貼り付けられた状態（テキストがぎっしり）
  - YAMLのプレビュー部分には面白そうなキーワードが見える：
    「style: editorial_collage」
    「background: grunge_texture, spray_paint」
    「icons: 3d_miniature (ramen, pottery, retro_game)」
    「layout: asymmetric_magazine」
    「hook: 『実質飲み会のゴルフ』」
  - YAMLの上部に「この内容で画像を生成してください」の1文を追加した状態
  - 入力欄全体をオレンジの枠で強調
- 下部：「YAMLを貼り付け＋\n『画像を生成してください』と書く」（黒・小さめ・中央揃え）

【サブステップ②】
- ライトグレーの角丸カード（薄い水色の「任意」バッジをカード右上に配置）
- 上部：オレンジの丸に白数字「②」
- 中央：スマートフォン正面モックアップ
  - 画面内：入力欄左端の「+」ボタン（添付ボタン）をオレンジの枠で強調
  - その上にメニューが開き「写真を選択」のオプションが見えている
  - 自分の顔写真の小さなサムネイルが添付された状態
- 下部：「（任意）\n自分の写真を添付すると\nカードに組み込まれる」（黒・小さめ・中央揃え）

【サブステップ③】
- ゴールドの細枠線付き角丸カード（完成を示す特別扱い）
- 上部：グリーンの丸に白チェックマーク「完成」
- 中央：スマートフォン正面モックアップ
  - 画面内：生成された図解プロフィールカードの画像が大胆に表示されている
  - 画像の周囲にゴールドのキラキラエフェクト

  《画像内のプロフィールカードの描写・重要》
  - レイアウト：雑誌のカバー／ストリートアート・コラージュ風のエディトリアルデザイン。均等グリッドレイアウト・ビジネステンプレート風は絶対に使わない
  - 背景：グランジ／ストリートアート調のテクスチャ（段ボール・スプレーペイント・破れた紙のコラージュ・マスキングテープが重なり合う質感）
  - 配色：多彩で大胆（蛍光ピンク・ライムグリーン・ビビッドイエロー・ブラックなど異なるアクセントカラーが同居）
  - タイポグラフィ：雑誌の特集タイトルのような極太フォント・スプレー風文字・スタンプ風文字が混在
  - 3つの情報セクションが**異なるサイズで非対称に配置**される（均等ではない）
  - 各セクションには3Dミニチュア風の固有アイコン（ラーメン丼・陶器・レトロゲーム機・観葉植物・ビールジョッキなど、具体的で個性的なオブジェクト）
  - セクション内のテキストには「実質飲み会のゴルフ」「限定製菓」など、思わずツッコミたくなる固有の言い回しが記載されている
  - 上部に大きく「キャッチコピー＋名前」（名前の横に「へなP」のようなニックネーム）
  - 写真（添付した場合）はコラージュの一部として中央下部に自然に溶け込む

- 下部：「送信→画像が届いたら\nプロフィールカード完成！」（黒・小さめ・中央揃え）

※厳守事項：
- 完成画像は『目を見張るデザイン』『面白い内容』を見せることが最重要。単調な青ベースの均等グリッド・トロフィーや歯車などの汎用ストックアイコン・「最適な計画を立案」のような無難なビジネス定型文は絶対に使わない
- 雑誌の特集ページや都市部の広告ビジュアルのような、スクロール中に思わず手を止める視覚的インパクトを持たせる

画像最下部（全幅）：
ブランドブルーの帯に白太字テキスト「完成！あなただけの図解プロフィールカードが今日できました。」
```
