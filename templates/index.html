📦 Prompt Vault — 使い方ガイド

このツールは、AIプロンプトの作成・編集・バージョン管理を一元化するツールです。
画面の見方から GitHub 連携まで、すべてここで確認できます。

🖥️ 画面の見方

エリア 説明
左サイドバープロンプトの一覧。クリックで開く。「＋新規プロンプト」で新規作成。
👁 表示タブプロンプトの内容を読みやすく表示します（Markdown形式で整形）。
✏️ 編集タブプロンプトを直接編集できます。Ctrl+S で保存。
🕐 履歴タブ過去に保存したバージョンの一覧。「👁 見る」で内容確認、戻すことも可能。
± 差分タブどこが変わったか色つきで確認できます（緑＝追加、赤＝削除）。
下のバーコメントを入力して「💾 保存」でバージョンを記録。Push/Pullでクラウドと同期。

📝 基本的な使い方

1

プロンプトを開く

左サイドバーのファイル名をクリック → 右側にプロンプトが表示されます。

2

プロンプトを編集する

「✏️ 編集」タブをクリック → 直接テキストを書き換えます。
編集中に Ctrl+S を押すとファイルが保存されます（まだバージョンは記録されません）。

3

バージョンを記録（保存コミット）

画面下のコメント欄に変更内容を一言で書いて（例：Q3に複数キャラ対応を追加）、
「💾 保存」ボタンを押します。これがバージョン記録（コミット）です。

4

過去に戻す

「🕐 履歴」タブ → 戻りたいバージョンの「👁 見る」→ 「このバージョンに戻す」。

🗂️ バージョン管理って何？

たとえ話： Wordの「上書き保存」は前のデータが消えてしまいます。
バージョン管理は「名前をつけて保存」を自動でやってくれる仕組みです。
「どんな変更をしたか」「いつ変更したか」が全部残るので、いつでも過去に戻れます。
  • ✅ 「誤って大事な部分を削除してしまった」→ 前のバージョンに戻せる
  • ✅ 「V1とV2のどこが違うか確認したい」→ 差分タブで色つき表示
  • ✅ 「なぜこの変更をしたか後から分かる」→ コメットに理由を書いておく

☁️ GitHub って何？

たとえ話： このツールのバージョン管理はあなたのPC内だけで動いています。
GitHubはそのデータをインターネット上（クラウド）に保存するサービスです。
「PCが壊れてもデータが消えない」「別のPCからでも使える」ようになります。
操作 意味
⬆ PushPCのデータをGitHubに送信（バックアップ＆共有）
⬇ PullGitHubのデータをPCに取得（別PCで更新した内容を同期）
リポジトリファイルやバージョン履歴をまとめて保存する「棚」のこと
コミット変更にコメントをつけて記録すること（このツールの「保存」ボタン）

🔗 GitHub に接続する手順

1

GitHubアカウントを持っているか確認

お使いのリポジトリ: https://github.com/henachoko-se/Prompt-Vault
✅ すでにリポジトリは作成済みです！

2

Personal Access Token（PAT）を取得する

PATとは、GitHubへのアクセスを許可する「鍵」のようなものです。
① GitHubにログイン → 右上のアイコン → Settings
② 左メニュー最下部 → Developer settings
③ Personal access tokens → Tokens (classic)
④ Generate new token (classic) をクリック
⑤ Note（メモ）: PromptVault と入力
⑥ Expiration（有効期限）: No expiration または任意の期限
⑦ スコープ: repo にチェック ✅
⑧ Generate token → 表示されたトークン（ghp_xxxx）をコピー

3

このツールで設定する

右上の「⚙ GitHub 未設定」ボタンをクリック
→ リポジトリURL と コピーしたToken を入力 → 「保存」

4

Push してみる

下のフッターの「⬆ Push」ボタンを押すと、GitHubにデータが送信されます。
はじめての Push 後は https://github.com/henachoko-se/Prompt-Vault で確認できます。

❓ よくある質問

💾「保存」と Ctrl+S の違いは？

Ctrl+S → ファイルをPCに書き込むだけ（バージョン記録なし）
フッターの「💾 保存」→ コメントをつけてバージョンとして記録（履歴に残る）
編集後は必ず「💾 保存」まで押してください。

📤 Push はいつ押せばいい？

「💾 保存」（コミット）した後に Push するとGitHubに反映されます。
毎日の作業終わりに Push する習慣をつけると安心です。

📁 新しいプロンプトを追加するには？

左サイドバーの「＋ 新規プロンプト」→ フォルダ選択 → ファイル名入力 → 「作成」

🔄 過去のバージョンに戻したい

対象プロンプトを開く → 「🕐 履歴」タブ → 戻りたいバージョンの「👁 見る」→「このバージョンに戻す」
※ 戻した後は忘れずに「💾 保存」でコミットしてください。

このガイドはいつでも左サイドバーの「❓ 使い方」から確認できます
