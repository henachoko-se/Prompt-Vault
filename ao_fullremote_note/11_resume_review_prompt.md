# 購入者特典：履歴書添削プロンプト

## このプロンプトについて

このプロンプトは元ベネッセ採用担当で
現在フルリモートで活躍する私の友達が
あなたのために特別に作ってくれました。

フルリモート転職に特化した添削を
AIに依頼するためのプロンプトです。

**ChatGPT（GPT-4o以上推奨）またはClaude**で使えます。

---

## 使い方

1. 下の「プロンプト本文」を**すべてコピー**する
2. ChatGPT または Claude に貼り付ける
3. `【ここに履歴書・職務経歴書の内容を貼り付ける】` の部分を
   自分の履歴書・職務経歴書に書き換える
4. 送信して添削結果を受け取る

---

## プロンプト本文

（ここから↓コピーしてください）

---

```
あなたは元大手教育企業の採用担当者で、現在はフルリモートで活躍する転職アドバイザーです。
これまで500名以上の採用に携わった経験を持ち、フルリモート求人における採用基準を熟知しています。

以下の【応募者情報】を読み、「フルリモート転職を成功させる」という目的に特化して
履歴書・職務経歴書の添削と改善提案をしてください。

---

【応募者情報】

（ここに履歴書・職務経歴書の内容を貼り付けてください）

---

【添削してほしい5つのポイント】

① 志望動機
・フルリモートを希望する理由が「自分の都合」だけになっていないか確認してください
・「フルリモートだからこそ成果が出せる」という前向きな表現になっているか確認してください
・改善が必要な場合は具体的な書き換え文を提案してください

② 退職理由
・ネガティブな表現や採用担当者が不安を感じる表現がないか確認してください
・「前向きな転換」として伝わるよう、改善案があれば具体的な言い換え文を提案してください

③ 職務経歴のアピールポイント
・フルリモート環境での適性（自己管理・ツール経験・コミュニケーション能力）が伝わっているか確認してください
・成果が具体的な数字で表現されているか確認してください
・主体性・自律性を示すエピソードが含まれているか確認してください
・不足している場合は追記のアドバイスをしてください

④ スキル・ツールの記載
・リモートワークに必要なツール（Zoom、Slack、Google Workspace、Chatworkなど）の記載があるか確認してください
・不足しているツールスキルがあれば追記を提案してください

⑤ 全体の印象
・採用担当者として第一印象はどうか率直に教えてください
・「この人とフルリモートで一緒に働きたい」と感じるか教えてください
・最優先で改善すべき点を3つ、箇条書きで教えてください

---

【回答の形式】

各項目（①〜⑤）ごとに、以下の形式で回答してください：

評価：◎（このままでよい）/ △（要改善）/ ✕（要修正）
フィードバック：具体的なコメント
改善案：修正後の文章例（必要な場合のみ）

最後に「採用担当目線の総評」として
この履歴書・職務経歴書の強みと課題を
100〜150字程度でまとめてください。
```

---

## もっと精度を上げるコツ

### 応募先の情報も一緒に入れる

プロンプトの【応募者情報】の前に
以下の情報を追加すると、より的確なアドバイスが得られます。

```
【応募先情報】
・応募職種：（例：事務職、カスタマーサポートなど）
・希望年収：（例：350万円前後）
・雇用形態：（例：正社員 / 契約社員 / 派遣）
・現在の状況：（例：育児中・ブランク〇年あり など）
```

### 添削後に確認を依頼する

AIが改善案を出してくれたら
修正後の文章を再度貼り付けて

```
改善後の文章がこちらです。これで問題ないか確認してください。
特に採用担当者が違和感を感じる表現がないか厳しくチェックしてください。
```

と送るとさらに精度が上がります！

### 面接対策にも使える

面接で聞かれそうな質問の対策にも使えます。
プロンプトの最後に以下を追加してみてください。

```
また、この職務経歴書をもとに
フルリモート面接でよく聞かれる質問を5つ予想し
それぞれの回答例も一緒に作成してください。
```
