# 履歴書・職務経歴書の添付方法 手順説明画像生成プロンプト

**対応特典:**  
- [購入者特典：履歴書添削プロンプト](./11_resume_review_prompt.md)
- [フルリモート転職特化・職務経歴書チェックプロンプト](./11b_shokumu_keikisho_prompt.md)

**用途:**  
AI初心者向けに、「プロンプトを送ったあと、どこで履歴書や職務経歴書を送ればいいか」を説明する画像。

---

## 仕様

- **形式:** 縦長（SNS投稿・note挿入向け）1枚完結版
- **推奨比率:** 9:16 または 1080×1920px
- **カラー:** ネイビー（#1A3A6B）×オレンジ（#E87722）×白
- **テイスト:** 初心者向け・やさしい・3ステップで迷わない・画面操作が想像できる
- **目的:** 「プロンプトを送るタイミング」と「書類を送るタイミング」を混同しないようにする

---

## 縦長9:16・1枚完結版

```
縦長9:16の手順説明インフォグラフィック画像を生成してください。
日本語テキストをすべて正確に表示してください。

【全体カラー】
- 背景：白
- メイン見出し帯：ネイビー（#1A3A6B）・白文字
- アクセント：オレンジ（#E87722）
- サブ背景：薄いネイビー、薄いオレンジ、薄いグレー
- 文字：濃いネイビー、重要語のみオレンジ

【上部タイトル帯】
背景：ネイビー
大タイトル（白文字・太字）：
「AIに履歴書・職務経歴書を送るやり方」
サブタイトル（白文字）：
「この3ステップでOK」

【セクション1：ステップ1】
見出し：
「1. まず、プロンプト本文を送る」
内容：
「購入者特典のプロンプト本文をコピー」
「ChatGPT・Claudeの入力欄に貼り付けて送信」
注意書き（オレンジ）：
「※まだ履歴書は送らなくてOK」
アイコン：
コピー、送信、チャットのアイコン

【セクション2：ステップ2】
見出し：
「2. AIからこの返事が来る」
チャット吹き出し風の枠に以下を表示：
「履歴書をこちらに貼り付けてください」
「テキスト・画像・PDF・Wordのどれでも大丈夫です」
注意書き（オレンジ）：
「この返事が出たら、次に書類を送ります」

【セクション3：ステップ3】
見出し：
「3. ここで書類を送る」
補足：
「やりやすい方法を1つ選べばOK」

3つの横長ボックスを縦に並べる。

ボックスA：
左に「A」
本文：
「文章をコピーできる → 入力欄にそのまま貼る」

ボックスB：
左に「＋」
本文：
「PDF・Wordがある → ＋ / クリップから添付」

ボックスC：
左に「写」
本文：
「紙・スマホ画面だけ → 写真やスクショを添付」

【セクション4：送る時に一緒に書く言葉】
見出し：
「送る時に一緒に書く言葉」
薄いグレーの入力欄風ボックス：
「添付した書類を添削してください。」
「読みにくい部分があれば教えてください。」

【下部結論帯】
背景：ネイビー
テキスト：
「迷ったら『文章を貼る』か『写真を送る』でOK」
サブテキスト：
「住所・電話番号など見せたくない情報は隠して大丈夫です」

【デザイン指定】
- AI初心者でも不安にならない、やさしい雰囲気
- 「まだ履歴書は送らなくてOK」と「この返事が出たら書類を送る」を強調する
- 入力欄、吹き出し、＋/クリップ添付を連想できる見た目にする
- 文字量を絞り、見出しだけで流れが分かる構成
- カードの角丸は控えめ
- 日本語テキストは省略・文字化け・誤字なく表示する
- note記事内に挿入しても読みやすい余白を確保する
```

## 生成後のチェックリスト

- [ ] タイトルで「何の説明画像か」がすぐ分かる
- [ ] 3ステップ（プロンプト送信→AIの返事→書類送信）になっている
- [ ] 「まだ履歴書は送らなくてOK」が入っている
- [ ] 「この返事が出たら書類を送る」が入っている
- [ ] 3つの送信方法（テキスト・ファイル・写真）が入っている
- [ ] 個人情報は隠してもOKと分かる
- [ ] 日本語テキストが正確に読める
- [ ] ネイビー×オレンジの配色になっている
