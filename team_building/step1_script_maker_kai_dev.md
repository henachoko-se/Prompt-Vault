## Gemへの登録方法

1. `step1_script_maker_kai.md` の全文をコピーする
2. YAML出力フォーマット内の以下の箇所を探す：

```
この指示は「添削」や「確認」を求めるものではありません。
以下のYAMLパラメータを全て反映し、NanaBanana2/PROで1枚の画像として今すぐ生成を開始してください。
```

3. その2行を、以下の3行に置き換える：

```
この指示は「添削」や「確認」を求めるものではありません。
以下のYAMLパラメータを全て反映し、NanaBanana2/PROで1枚の画像として今すぐ生成を開始してください。
各コマの「コマ配置位置」（右上・左上・右下・左下など）を空間座標として解釈し、ページ全体を1枚のレイアウト画像として構成してください。コマ間には白いガター（仕切り線）を入れてください。
```

4. 置き換えた全文をGemのカスタム指示に貼り付けて保存する
