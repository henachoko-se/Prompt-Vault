# 【図解1】比較インフォグラフィック「図解が刺さらない真の原因は"場所"」

**挿入位置：** 「なぜ『図解の場所』を決めるのが、こんなに難しいのか」セクション末尾
**目的：** 「デザイン vs 場所」という問題の切り分けを瞬時に伝える
**形式：** 横長 16:9 / 左右比較インフォグラフィック

```
横長（16:9）のインフォグラフィック画像を生成してください。

【全体レイアウト】
- 背景：薄いクリーム色（暖色系パステル）
- 上部：タイトル帯（テラコッタオレンジ背景・白文字）
- 中央：左右2分割
- 下部：結論メッセージ帯（セージグリーン背景・白文字）

【タイトル帯】
大タイトル（白文字・太字）：「図解が刺さらない真の原因」
サブタイトル（白文字・細字）：「デザインじゃない。"場所と型のミスマッチ"です。」

【中央：左右比較】

■ 左側（薄いグレー＋ペールピーチの優しい背景）
カードタイトル：「みんなが思っている原因」
チェック項目（控えめなグレー文字・▲マーク）：
- ▲ 絵心がない
- ▲ 画像生成プロンプトが下手
- ▲ おしゃれなツールを知らない
キャプション（小さく）：「ここに労力を注いでも、刺さらない図解は変わらない」

■ 右側（暖かいクリーム背景・テラコッタ強調枠）
カードタイトル：「実際の原因（9割がここ）」
チェック項目（テラコッタ文字・⭕マーク）：
- ⭕ 入れる場所のミスマッチ
- ⭕ 文章構造に合わない型を選んでいる
- ⭕ 上流の"設計判断"を人がやっている
キャプション（強調）：「= 『図解しおり』が解決する領域」

【下部：結論帯】
横長の帯（セージグリーン背景・白文字・太字）
中央メッセージ：「図解は"作る"より"場所決め"が9割。」

【デザイン指定】
- 左右の対比は穏やかなパステルトーンで（攻撃的にならない）
- 角丸・薄い影・余白を十分に取る
- 全体的に温かみのある雰囲気
- 日本語テキストはすべて正確に表示
```

# 【図解2】8型チートシート「これだけ覚えれば80%カバー」

**形式：** 横長 16:9 / 8カード一覧（2行×4列）

```

【中央：8カード（2行×4列）】

- 角丸長方形・白背景・薄いペールピーチの枠線
```

# 【図解3】Before/Afterデモ「長文 → 図解しおりマップ」

**挿入位置：** 「出力イメージ：これが図解しおりマップです」セクション末尾
**目的：** 長文を投げ込んだ瞬間に図解配置マップが返ってくる、その「Wow」を視覚化
**形式：** 横長 16:9 / 左右比較（Before：長文の塊 / After：マップ化後）

```
横長（16:9）のインフォグラフィック画像を生成してください。

【全体レイアウト】
- 背景：薄いクリーム色（暖色系パステル）
- 上部：タイトル帯（テラコッタオレンジ背景・白文字）
- 中央左：Before（長文）
- 中央：「→」大矢印
- 中央右：After（図解配置マップ）
- 下部：所要時間帯（セージグリーン背景・白文字）

【タイトル帯】
大タイトル（白文字・太字）：「Before / After：図解しおりが、長文をマップに変える」

【中央左：Before】
- ラベル：「Before：図解の入れ場所が分からない長文」（グレー帯）
- 表現：A4文書をモチーフにした白い紙のグラフィック。
  紙の上に細かい横線で大量の文字が書かれているように表現
  （実際の文字ではなく、ぱっと見でテキストの密度が分かるダミーライン）。
- 紙の右側に大きな「？」マークを薄いグレーで配置（どこに図解？の問い）。

【中央：→大矢印】
テラコッタオレンジの太い矢印（「→」）
矢印の上下に小さく「図解しおりが解析」「8型で判定」のラベル

【中央右：After】
- ラベル：「After：📍マーカー付き配置マップ」（テラコッタ帯）
- 表現：同じA4文書のグラフィック。
  ただし4箇所に「📍」マーカーが付いており、
  それぞれの近くに小さなカードで型名がラベリングされている。
  - 📍 ポイント1：ステップ型
  - 📍 ポイント2：2カラム比較型
  - 📍 ポイント3：3ポイント型
  - 📍 ポイント4：チェックリスト型
- マーカーは小さなテラコッタの丸＋白の📍で表現。

【下部：所要時間帯】
横長の帯（セージグリーン背景・白文字・太字）
中央メッセージ：「長文を貼るだけ、5秒でマップ完成。10分後には図解4枚。」

【デザイン指定】
- BeforeとAfterは同じA4紙ベースで「変化」が分かるように
- マーカーは目立たせるが、紙の本文は読めない程度の抽象表現でOK
- 矢印は中央でしっかり視線誘導
- 全体的に温かみがあり、家庭でも気軽に使える雰囲気
- 日本語テキストはすべて正確に表示
```
