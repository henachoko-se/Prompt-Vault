# 【図解1】比較インフォグラフィック「図解が刺さらない真の原因は"場所"」

**挿入位置：** 「科学的メカニズム：認知負荷理論と場所と型のミスマッチ」セクション末尾
**目的：** 「デザイン vs 場所」という問題の切り分けを瞬時に伝える
**形式：** 横長 16:9 / 左右比較インフォグラフィック

```
横長（16:9）の温かみのあるインフォグラフィック画像を生成してください。

【全体レイアウト】
- 背景：クリーム（#FBF6E9）
- 上部：タイトル帯（テラコッタ背景・白文字）
- 中央：左右2分割
- 下部：結論メッセージ帯（セージグリーン背景・白文字）

【タイトル帯】
大タイトル（白文字・太字）：「お便りが読まれない真の原因」
サブタイトル（白文字・細字）：「文章じゃない。"場所と型のミスマッチ"です。」

【中央：左右比較】

■ 左側（薄いペールピーチ背景・「思い込み」ラベル）
カードタイトル：「みんなが思っている原因」
チェック項目（グレー文字・▲マーク）：
- ▲ 文章が長すぎる
- ▲ 字が多くて読みにくい
- ▲ 配るタイミングが悪い
キャプション（小さく）：「ここに労力を注いでも、読まれないお便りは変わらない」

■ 右側（明るいクリーム背景・テラコッタ強調枠・「真の原因」ラベル）
カードタイトル：「実際の原因（9割がここ）」
チェック項目（テラコッタ文字・⭕マーク）：
- ⭕ 図解の入れ場所のミスマッチ
- ⭕ 文章構造に合わない型を選んでいる
- ⭕ 上流の"設計判断"を人間がやっている
キャプション（強調）：「= 『図解しおり』Pro版が解決する領域」

【下部：結論帯】
横長の帯（セージグリーン背景・白文字・太字）
中央メッセージ：「お便りは"作る"より"場所と型の判断"が9割。」

【デザイン指定】
- 左右の対比は穏やかなトーンで表現
- 家庭の文書に貼っても違和感のない優しいトーン
- 角丸・薄い影・余白を十分に取る
- 日本語テキストはすべて正確に表示
```

# 【図解3】Before/Afterデモ「長文 → 図解しおりマップ＋画像1枚目」

**挿入位置：** 「出力イメージ：図解しおりマップ＋画像生成までワンストップ」セクション末尾
**目的：** 長文を投げると「No.〇〇 〇〇型」のマップ＋画像1枚目までワンストップで返ってくるWowを視覚化
**形式：** 横長 16:9 / 左右比較（Before：長文の塊 / After：マップ＋生成画像）

```
横長（16:9）の温かみのあるインフォグラフィック画像を生成してください。

【全体レイアウト】
- 背景：クリーム（#FBF6E9）
- 上部：タイトル帯（テラコッタ背景・白文字）
- 中央左：Before（長文の塊）
- 中央：「→」大矢印
- 中央右：After（マップ＋画像1枚目）
- 下部：成果メッセージ帯（セージグリーン背景・白文字）

【タイトル帯】
大タイトル（白文字・太字）：「Before / After：長文 → マップ＋画像1枚目までワンストップ」

【中央左：Before】
- ラベル：「Before：1500字のPTAお便り」（グレー帯）
- 表現：A4文書をモチーフにした白い紙のグラフィック。
  紙の上に細かい横線で大量の文字が書かれているように表現
  （実際の文字ではなく、ぱっと見でテキストの密度が分かるダミーライン）。
- 紙の右側に大きな「？」マークを薄いグレーで配置（どこに図解？の問い）。

【中央：→大矢印】
テラコッタ色の太い矢印（「→」）
矢印の上下に小さく「図解しおりPro版が解析」「68パターンから自動選定」のラベル

【中央右：After】
- 上半分：「After：マップ出力（重要度順）」（テラコッタ帯）
  - 同じA4文書のグラフィック。
    ただし複数箇所に「🔖」しおりマーカーが付いており、
    それぞれの近くに小さなカードで「No.〇〇 〇〇型」とラベリング。
    - 🔖 ポイント1：No.2 3ポイント型
    - 🔖 ポイント2：No.14 ステップ型
    - 🔖 ポイント3：No.45 YES/NO診断型
    - 🔖 ポイント4：No.66 保存推奨型
  - マーカーは小さなテラコッタの丸＋白の🔖で表現。
- 下半分：「画像1枚目（自動生成）」のミニチュア
  - 3カードが横並びになった「3ポイント型」のサムネ画像（暖色パステル）
  - 「実際にAIが生成した1枚目」のキャプション

【下部：成果メッセージ帯】
横長の帯（セージグリーン背景・白文字・太字）
中央メッセージ：「長文を貼るだけ、数秒でマップ＋画像1枚目までワンストップ。」

【デザイン指定】
- BeforeとAfterは同じA4紙ベースで「変化」が分かるように
- マーカーは目立たせるが、紙の本文は読めない程度の抽象表現でOK
- 「No.〇〇 〇〇型」の番号付きラベルが視認できる程度の大きさで
- 矢印は中央でしっかり視線誘導
- 家庭の文書に貼っても違和感のない温かいトーン
- 日本語テキストはすべて正確に表示
```
