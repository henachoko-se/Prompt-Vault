**実施日:** 2026-05-22  
**対象:** `family_building_science/vol004/prompt.md`  
**目的:** お土産プロンプト冒頭の9つのパラメータが、表示だけでなく回答内容に正しく反映されるかを確認する  
**方法:** プロンプト仕様の静的レビュー、想定入力による内部シミュレーション、パラメータ変更時の期待挙動チェック

## 結論

🟢 **V4合格**

9つのパラメータは、V4修正後はすべて回答方針に反映される設計になっています。

今回のテストで、V3時点では以下の4項目が「起動時に表示はされるが、変更時の反映ルールが明文化不足」でした。

- `DEFAULT_EXPERIENCE_LEVEL`
- `DEFAULT_ADVICE_DEPTH`
- `DEFAULT_UNIT_STYLE`
- `DEFAULT_CONFIRMATION_POLICY`

そのため、`prompt.md` に **「パラメータ反映ルール」** を追加し、9項目それぞれが回答へどう効くかを明記しました。

`フェーズ0` の直後に、以下の観点を明記しました。

| パラメータ | V4で明文化した反映内容 |
|---|---|
| `DEFAULT_REGION` | 種まき・植え付け・収穫時期、暑さ寒さ、梅雨・霜・真夏の水切れなどへ反映 |
| `DEFAULT_GROWING_STYLE` | 地植え、プランター、鉢、室内、ベランダごとに水やり・土量・日当たり・風通し・土づくり手順を変更。小規模プランターで元の土がない初心者には野菜用培養土を優先 |
| `DEFAULT_EXPERIENCE_LEVEL` | 初心者はやさしく具体的に、中級者はpH・施肥設計・病害虫候補まで少し詳しく |
| `DEFAULT_SOIL_MATERIALS` | 優先資材として使うが、全部そろえさせず代替案を出す |
| `DEFAULT_OUTPUT_MODE` | 出力モード別の調整ルールへ接続 |
| `DEFAULT_ADVICE_DEPTH` | 超短く / やさしく具体的 / 詳しめ の出力密度を変更 |
| `DEFAULT_SAFETY_LEVEL` | 食用作物、薬剤、子ども・ペット、誤食、肥料の扱いで、どの注意を優先して出すかを変更。「家庭菜園向け安全優先」「薬剤なし優先」「子ども・ペット配慮強め」などを選ぶ |
| `DEFAULT_UNIT_STYLE` | g・ml中心か、直感的な目安併記かを出力に反映 |
| `DEFAULT_CONFIRMATION_POLICY` | 質問数を制御し、少ない質問方針なら仮説を置いて進める |

## 1. 単体パラメータテスト

### P-01 `DEFAULT_REGION`

**入力:** `DEFAULT_REGION = 北海道`、相談「5月にミニトマトを植えていい？」  
**期待:** 温暖地前提にせず、霜・低温・定植時期の遅れを説明する。  
**結果:** 🟢 PASS  
**確認点:** 地域差ルールとパラメータ反映ルールの両方で支えられる。

**入力:** `DEFAULT_REGION = 沖縄県`、相談「夏の水やりが不安」  
**期待:** 高温、強日差し、台風、乾きやすさに触れる。  
**結果:** 🟢 PASS

### P-02 `DEFAULT_GROWING_STYLE`

**入力:** `DEFAULT_GROWING_STYLE = プランター栽培`、相談「土づくりしたい」  
**期待:** 土量、鉢底から流れる水やり、乾きやすさを反映。元の土がない小規模プランターなら、まず野菜用培養土を優先し、自作する場合だけ赤玉土7：バーク堆肥3を補足する。  
**結果:** 🟢 PASS

**入力:** `DEFAULT_GROWING_STYLE = 地植え`、相談「土づくりしたい」  
**期待:** 畝、排水、既存土、1平方メートルあたりの目安、寝かせる期間を反映。  
**結果:** 🟢 PASS

## 5. テストで確認した出力イメージ

### 例1：地域と育て方だけ変更

**設定:**  
`DEFAULT_REGION = 神奈川県`  
`DEFAULT_GROWING_STYLE = ベランダ・プランター栽培`

**期待される出力の変化:**
- ベランダの乾きやすさ、照り返し、風通しを確認する
- 地植え前提の畝づくりではなく、鉢底から流れる水やり、土量、プランター向けの説明へ寄せる
- 元の土がない小規模プランターの場合は、まず野菜用培養土を優先する
- 自作配合をしたい場合だけ、赤玉土7：バーク堆肥3を土台として補足する

### 例2：土づくり重点 + 詳しめ

**設定:**  
`DEFAULT_OUTPUT_MODE = 土づくり重点`  
`DEFAULT_ADVICE_DEPTH = 詳しめ`

**期待される出力の変化:**
- 目標pH、排水性、保水性、肥沃度を説明
- 基本4資材と発展3資材を分ける
- 石灰、堆肥、油かす・米ぬか、化成肥料の投入時期を分ける
- 寝かせる期間と入れすぎリスクを説明

### 例3：今日の一手だけ + 初心者

**設定:**  
`DEFAULT_OUTPUT_MODE = 今日の一手だけ`  
`DEFAULT_EXPERIENCE_LEVEL = 初心者`

**期待される出力の変化:**
- 今日やることを最大3つに限定
- 専門用語を避ける
- 「今日はやらなくていいこと」を明記
- 次に見るポイントを1〜3個に絞る

## 残存リスク

| リスク | 評価 | 対応 |
---|---|---|
| 生成AIによっては9項目をすべて毎回強く反映しようとして回答が長くなる | 低 | `DEFAULT_OUTPUT_MODE` と `DEFAULT_ADVICE_DEPTH` で抑制可能 |
| ユーザーが設定値を自由文で書いた場合、解釈に揺れが出る | 低 | 起動メッセージに例を提示済み |
| 写真が不鮮明な場合、パラメータ以前に観察精度が下がる | 中 | 断定回避と追加写真依頼でカバー |
| 土壌pHや肥料量は実測なしでは目安になる | 中 | 「目安」「候補」として出す設計でカバー |

## 最終判定

Vol.004の9つのパラメータは、V4補強後、以下を満たしています。

- 起動時に現在設定として表示される
- 最低限直すべき項目が「地域」と「育て方」だと分かる
- 残り7項目は必要に応じて後から調整できる
- 各パラメータが回答内容へどう反映されるかが明文化されている
- 安全・断定回避・誇張回避の上位ルールを壊さない

**総合判定:** 🟢 **合格**
